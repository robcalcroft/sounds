This site uses Soundcloud's API ( application program interface ) to grab a playlist and then play through it.
The playlist URL is sent in a GET request and from the result I can get a list of track IDs. From there I can use the track ID to stream the tracks.

I also save the playlist and track number currently playing for you, so pick up where you left off. Aren't I nice!
